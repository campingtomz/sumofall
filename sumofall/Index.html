Sum Of All Fears Exercise

or Palindrome Exercise

  • This Exercise takes a user input and returns whether any two numbers from the list add up to it.
  • Example if the list contains [10, 15, 3, 7]
  • If the user would enter 17, then the program would return frue.
  • 10 + 7 = 17
  • If the user enters 14, the program returns false, as no combination of the list values = 14.
  • There are several ways to build an algorithm for this problem:
  • The brute force method would be to have a nested loop iterate throw the list.
  •                             
                            userinput = k;
                            for(i = 0; i < array.length; i++){
                                value = array[i];
                                for(k = i+1; k < array.length; k++){
                                    if(userinput - value === array[k]){
                                        return true;
                                    }
                                }
                            }
                            return false;
